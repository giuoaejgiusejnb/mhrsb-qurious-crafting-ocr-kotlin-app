mhrsbの練成データを画像認識し，欲しいスキルが入った画像のみを抽出します．  
アプリではcolabに渡すファイルをdriveにアップロードし，この[ipynbファイル](https://colab.research.google.com/github/giuoaejgiusejnb/mhrsb-qurious-crafting-ocr-kotlin-app/blob/main/notebooks/qurious_crafting_app.ipynb)
を開いて画像認識を行います．  
[ここ](https://github.com/giuoaejgiusejnb/mhrsb-qurious-crafting-ocr-kotlin-app/releases)からMHR-SB-.OCR.apkをダウンロードしてください  
androidの標準設定だと，Google Playストア以外からのアプリのインストールは制限されているので，  
「設定」⇒「アプリ」または「アプリと通知」⇒「特別なアプリアクセス」⇒「不明なアプリのインストール」⇒Filesを選択して「この提供元を許可」  
を指定してください．
